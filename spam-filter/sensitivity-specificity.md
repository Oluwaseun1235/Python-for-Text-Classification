#### True positive
- Text is spam
- Classifier predicts spam

#### True negative
- Text is not spam
- Classifier predicts not spam

#### False positive
- Text is not spam
- Classifier predicts spam

#### False negative
- Text is spam
- Classifier predicts not spam
